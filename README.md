# Human Resources Analytics Dashboard

## Project Overview
This project focuses on analyzing Human Resources data using Power BI to generate meaningful workforce insights and support better decision-making.

## Features
- Employee Performance Analysis
- Recruitment Metrics Analysis
- Employee Turnover Insights
- Workforce Data Visualization
- Interactive Dashboard Design

## Tools Used
- Power BI
- Data Cleaning & Transformation
- Data Visualization
- Microsoft Excel / CSV Dataset

## Project Files
- HR_Analytics.xlsx.csv (Dataset)
- HR analytics dashboard.pbix (Power BI Dashboard File)

## Objective
The objective of this project is to analyze workforce metrics, identify trends, and create interactive dashboards for HR decision-making.
